Project Lyrae
=============
API Server powered by Node.js   


## GET /users ##
**List all users**   

## GET /user/:id ##
**Get an user data given :id**   

## POST /user ##
**Create an user given :id & :name**   

    id=\<fbid\>   
    name=\<user name\>   
    
## POST /user/:id ##
**Modify an user attribute given :id**   

    lng=\<longitude\>   
    lat=\<latitude\>   
    activity=\<activity\>   
    matching=\<boolean\>   
    
## GET /match/:id ##
**Match an user given :id by its activity**
